# Capstone_Project
This is my code for my senior year data science capstone project that uses neural network machine learning in order to classify stock signals from buy, sell, and hold in order to create a proffitable trading algorithm. 

To use this code follow the following steps in your terminal (only copy and paste bolded letters into your terminal):
1. **python --version** (checks if you have the proper version of python installed)
2. **git clone https://github.com/jbandreski/Stock_Signal_Classification.git** (clones this git repository to your computer)
3. **cd Stock_Signal_Classification** (puts you into the git repository)
4. **python -m venv venv** (creates a virtual environment for you to safely run the code)
5. **source venv/bin/activate** (start the virtual environment on mac or linux)
   **venv\Scripts\activate** (start the virtual environment on windows)
   when you see (venv) at the beginning of your terminal lines you know the virual environment has started
6. **pip install -r requirements.txt** (install the required libraries for you to run the program
7. **mkdir -p results** (creates a temporary results folder that will store the confusion matrix and the cumulative return charts)
8. **python main.py** (this will run the code)
To see the charts the code produces you will need to find the git repository on your computer and go into the results folder and open the charts manually.
